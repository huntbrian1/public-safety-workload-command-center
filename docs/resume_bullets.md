# Resume Bullets

## Technical Version

Built a public safety service-demand intelligence platform using Python, PySpark, Streamlit, and Excel-ready outputs to consolidate large-scale service-event records with weather, calendar, and geographic metadata into a cleaned analytics workflow.

## Business-Facing Version

Built a public safety operations analytics platform that identifies service-demand patterns, segments operating zones, surfaces high-volume periods, and translates workload indicators into resource-planning recommendations.

## Dashboard Version

Created a Streamlit workload command center with period filters, mapped beat coverage, zone-hour heatmaps, timing patterns, service-category mix, weather-context views, and methodology/data-quality notes.
